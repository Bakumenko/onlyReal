Оптимизация по AST-дереву №13

Замена узлов условных операторов, у которых в выражениях по обоим ветвям null, на узел с null.
Например:

Исходный код:         Оптимизированный:
    if(a>=a)           null;  
	  null;            
    else
      null;
Применение оптимизации:
var r = parser.root;                     //Построение AST-дерева
var opt13_visitor = new OptVisitor_13();   //Создание экземпляра класса оптимизации
r.Visit(opt13_visitor)                    //Обход дерева визитором с оптимизацией
