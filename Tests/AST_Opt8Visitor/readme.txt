Оптимизация по AST-дереву №8

Замена выражений, всегда возвращающих истину, вида: a==a, a>=a на true. Например:

Исходный код:         Оптимизированный:
    if(a>=a)           if(true)  
	    b = 10;            b = 10;

Применение оптимизации:
var r = parser.root;                     //Построение AST-дерева
var opt8_visitor = new OptVisitor_8();   //Создание экземпляра класса оптимизации
r.Visit(opt8_visitor)                    //Обход дерева визитором с оптимизацией
