Как собирать интерфейс:
	На уровне выше есть проект OptLib.csproj. Для начала следует собрать его (дело в том, что я делал интерфейс на 13 студии (какую нашел), а она не все поддерживает из того, что применяется в коде).
	После сборки получится библиотека OptLib.dll. Её следует положить в каталог OptimCompiler\GUI\OptGui. После этого можно открывать проект интерфейса и собирать его.





Чтобы открыть файл с программой есть кнопка - открыть. Основная вкладка программы разбита на два отдела, которые полностью равнозначные. Сверху находятся две combobox, в которых можно выбрать режим отображения кода. Вначале в списке идет вывод для оригинальной программы (трехадресный код, CFG, запуск, ...), затем идет список для оптимизированной программы почти то же самое. Во вкладке оптимизаций можно выбирать различные оптимизации, которые будут каскадно применены к исходному коду программы (справа ноходятся оптимизации по AST, а слева - для трехадресного кода)
От начала списка до кнопки запуска - это все для оригинального кода (без оптимизаций).
Далее для оптимизированной программы.
